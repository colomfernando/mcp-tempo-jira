# Work Attribute Tools

## `list_work_attributes`

List all work attributes defined in the Tempo workspace. Use this to discover available attributes and their valid values before creating or updating a worklog.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `offset` | number | No | Pagination offset (default: 0) |
| `limit` | number | No | Max results to return (default: 50) |
